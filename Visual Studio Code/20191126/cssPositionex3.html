CSS 포지셔닝

Fixed Position

Fixed 포지션은 항상 정해진 위치에 고정됩니다. 위쪽의 CSS 포지셔닝을 포함하는 nav 박스가 fixed 포지션으로 선언 되었습니다. 마우스 스크롤의 조작에 상관없이 화면 상단에 고정됩니다.

Absolute Position

앱솔루트 포지션으로 요소를 선언하면 부모요소의 위치로부터 포지션을 지정합니다. 부모요소는 항상 포지션이 relative 여야 합니다. 그렇지 않으면 상위의 Body 태그에 맞추게 됩니다.

앱솔루트 포지션 요소는 다른 요소가 패딩/마진에 영향을 받지 않습니다.

In this example, the red square has position: absolute, which means it will be absolutely positioned to it's parent container, which is the gray container. With the CSS to translate the box 50px from the top, and 20px from the left, this means the red box will move that much away from the top left corner of the gray container.

.absolute-plus-styles {
   position: absolute;
   top: 50px;
   left: 20px;
}

Relative Position

상대위치 포지션은 기존의 위치에서 좌표값만큼 이동합니다. 다른 요소들도 기존의 위치에 영향을 받습니다.

In this example, notice how we are using the same styles to translate our element like in the above example, but this element has been translated from it's relative position (see what I did there?)

.relative-plus-styles {
   position: relative;
   top: 50px;
   left: 20px;
}

Float and Clear

We can position elements to start from the left or right side of the window by using the float property. The float CSS property places an element on the left or right side of its container, allowing text and inline elements to wrap around it. The element is removed from the normal flow of the page, though still remaining a part of the flow (in contrast to absolute positioning).
